Setup / 05

SonarCloud configuration

Connect your SonarCloud account to pull project findings.

Generate a token at SonarCloud → My Account → Security.

Required to list SonarCloud projects. Find it in your SonarCloud organization URL or settings.

Setup / 06

Socket.dev configuration

Connect your Socket.dev organization to pull dependency risk findings.

Generate an organization token at Socket.dev → Organization Settings. Required scope: alerts:list. No repository write scopes are required.

Required to scope Socket.dev alerts during sync. Find it in your dashboard URL: https://socket.dev/dashboard/org/<org-slug>.

Setup / 07

Snyk configuration

Connect your Snyk account to import project vulnerabilities.

Create a personal or service account token at Snyk → Account Settings.

Required to scope Snyk issues during sync. Find it in your Snyk URL or Organization Settings.
